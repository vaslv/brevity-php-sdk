# Security Policy

## Supported versions

Only the latest release receives security fixes.

## Reporting a vulnerability

Please report vulnerabilities privately — do **not** open a public
issue:

- GitHub: use [private vulnerability reporting](https://github.com/vaslv/brevity-php-sdk/security/advisories/new), or
- Email: vaslv@proton.me

Include a description, reproduction steps and the affected
version/commit. You will get an acknowledgement within a few days; a
fix or mitigation plan follows as soon as the report is validated.
